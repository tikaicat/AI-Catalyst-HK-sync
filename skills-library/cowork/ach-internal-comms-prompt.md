# ACH Internal Communications Skill - Trigger Prompt

## 🎯 Skill Purpose
規劃和創建內部溝通材料，確保團隊對齊

## 📌 Trigger Scenarios

### When to Use
1. **公司公告** - 需要發佈重要的公司消息
2. **項目更新** - 定期分享項目進展
3. **政策文檔** - 建立或更新內部政策
4. **培訓材料** - 為員工創建培訓內容

## 💬 Sample Trigger Prompts

### Scenario 1: Company Announcement
```
使用 ach-internal-comms：
請建立公司公告 [主題]：
- 背景信息
- 關鍵信息點
- 受影響的團隊或流程
- 需要的行動
- 聯繫人信息
- 發佈渠道
```

### Scenario 2: Project Status Update
```
使用 ach-internal-comms：
為 [項目名稱] 創建進度更新：
- 完成的里程碑
- 當前焦點
- 遇到的挑戰
- 下週/月的計劃
- 需要的資源或協助
- 風險提醒
```

### Scenario 3: Internal Policy Document
```
使用 ach-internal-comms：
建立 [政策主題] 的內部政策文檔：
- 政策目標和原因
- 適用範圍
- 具體規則和指南
- 例外情況
- 實施時間表
- 問題提交流程
```

## 📋 Output Format

✅ 應該包含：
- 清晰的標題和摘要
- 結構化的內容
- 行動項目（如適用）
- FAQ 部分
- 聯繫信息
- 版本和發佈日期

## 🔗 Related Skills

- **tc-content-generator** - 內容擴展
- **ach-doc-coauthoring** - 審批流程

## 💡 Pro Tips
- 針對不同受眾調整溝通級別
- 提供多種語言版本（中文/英文）
- 使用清晰的摘要便於快速理解
- 提供額外資源的鏈接
