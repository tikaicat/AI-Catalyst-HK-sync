# TC Content Generator Skill - Trigger Prompt

## 🎯 Skill Purpose
為各種平台和媒介生成高質量的內容

## 📌 Trigger Scenarios

### When to Use
1. **社交媒體內容** - 為Facebook、Instagram、LinkedIn等生成帖子
2. **博客文章** - 撰寫SEO優化的博客內容
3. **營銷文案** - 創建產品描述和營銷信息
4. **電子郵件活動** - 生成新聞通訊和營銷電郵

## 💬 Sample Trigger Prompts

### Scenario 1: Social Media Content Series
```
使用 tc-content-generator：
請為 [主題] 生成一周的社交媒體內容：
- 平台：[Instagram/LinkedIn/Twitter等]
- 內容類型：[提示、技巧、新聞等]
- 發佈計劃：[每日]
- 品牌聲調：[專業/輕鬆/教育等]
- 包含的視覺元素：[圖片類型]
- 適用的標籤和主題標籤
```

### Scenario 2: Blog Article Series
```
使用 tc-content-generator：
為我們的博客創建關於 [主題] 的文章系列：
- 文章數量：[數量]
- 每篇文章長度：[字數]
- 目標受眾：[受眾描述]
- SEO 關鍵詞：[關鍵詞]
- 包含的部分：[標題、簡介、章節等]
- 行動呼籲：[CTA]
```

### Scenario 3: Marketing Campaign
```
使用 tc-content-generator：
為 [產品/服務] 創建營銷活動內容：
- 目標：[目標]
- 目標受眾：[受眾]
- 核心信息：[關鍵點]
- 內容格式：[文案, 電郵, 廣告等]
- 分佈渠道：[渠道]
- 性能指標：[KPI]
```

## 📋 Output Format

✅ 應該包含：
- 內容段落或帖子
- 標題和子標題
- 相關圖片/視頻建議
- 標籤和關鍵詞
- 發佈時間建議
- 替代版本（A/B 測試）
- 性能優化建議
- 後續跟進建議

- 包含視覺和互動元素
- 使用數據和統計支持主張
- 創建內容日曆和發佈計劃
- 監視和分析內容性能
