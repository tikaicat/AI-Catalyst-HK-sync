# Notion Template Builder Skill - Trigger Prompt

## 🎯 Skill Purpose
設計和構建 Notion 模板以實現高效的數據管理和團隊協作

## 📌 Trigger Scenarios

### When to Use
1. **項目管理模板** - 為項目跟蹤和管理創建Notion模板
2. **內容管理系統** - 建立內容資料庫和發佈工作流
3. **知識庫** - 組織和共享團隊知識
4. **客戶管理** - 建立CRM數據庫和流程

## 💬 Sample Trigger Prompts

### Scenario 1: Project Management Template
```
使用 notion-template-builder：
請為 [項目類型] 創建 Notion 項目管理模板：
- 項目屬性：[狀態, 優先級, 截止日期等]
- 任務跟蹤：[任務字段]
- 視圖需求：[看板, 日曆, 時間表]
- 自動化：[觸發器和操作]
- 報告需求：[報告類型]
```

### Scenario 2: Content Database
```
使用 notion-template-builder：
為 [內容類型] 建立 Notion 內容數據庫：
- 內容字段：[字段列表]
- 分類和標籤：[分類方式]
- 工作流狀態：[草稿, 審批, 發佈等]
- 相關內容鏈接
- 版本控制
- 發佈日期和計劃
```

### Scenario 3: Knowledge Base
```
使用 notion-template-builder：
創建團隊知識庫 Notion 模板：
- 知識類別：[類別]
- 搜索和標籤系統
- 文章/頁面結構
- 最後更新日期跟蹤
- 貢獻者信息
- 相關文章推薦
```

## 📋 Output Format

✅ 應該包含：
- 數據庫架構和字段定義
- 視圖配置（表格、看板、日曆等）
- 關係和链接配置
- 模板塊配置
- 數據庫模板
- 自動化規則
- 權限和共享設置
- 使用說明書
