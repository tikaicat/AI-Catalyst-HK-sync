# ACH Brand Guidelines Skill - Trigger Prompt

## 🎯 Skill Purpose
建立和維護品牌指南，確保所有內容和設計保持一致

## 📌 Trigger Scenarios

### When to Use
1. **新品牌建設** - 從頭建立品牌識別系統
2. **品牌審計** - 審查現有品牌使用的一致性
3. **擴展指南** - 為新產線或市場更新品牌指南
4. **培訓文檔** - 為團隊成員建立品牌培訓材料

## 💬 Sample Trigger Prompts

### Scenario 1: Brand Identity Creation
```
使用 ach-brand-guidelines：
請為 AI Catalyst HK 建立完整的品牌指南，包括：
- 品牌願景和使命陳述
- Logo 使用規則
- 顏色調色板（主色、輔助色）
- 字體選擇和層級結構
- Voice & Tone 指南
- 商標使用政策
```

### Scenario 2: Brand Consistency Audit
```
使用 ach-brand-guidelines：
對我們的品牌使用進行審計：
- 檢查現有內容的品牌一致性
- 識別偏離品牌指南的地方
- 建議改進方案
- 優先級排序改進項目
```

### Scenario 3: Brand Expansion Guide
```
使用 ach-brand-guidelines：
為 [新市場/新產品線] 建立品牌擴展指南：
- 本地化考慮因素
- 文化適應性調整
- 新顏色或設計元素（如適用）
- 應用指南更新
```

## 📋 Output Format

✅ 應該包含：
- 品牌演變歷史
- 核心品牌價值
- 視覺識別規則
- 語言指南
- 應用示例
- 不應該做的事項
- 常見問題解答

## 💡 Pro Tips
- 使用視覺示例而不僅是文字描述
- 建立可下載的品牌資產包
- 定期（每季度）審查和更新
- 為不同部門創建特定的應用指南
