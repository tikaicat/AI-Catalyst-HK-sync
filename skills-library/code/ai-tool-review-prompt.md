# AI Tool Review Skill - Trigger Prompt

## 🎯 Skill Purpose
評估和審查AI工具、模型和集成的功能、性能和適用性

## 📌 Trigger Scenarios

### When to Use
1. **工具評估** - 評估新的AI工具是否適合項目
2. **性能基準測試** - 比較不同工具的性能
3. **集成兼容性** - 檢查工具與現有系統的兼容性
4. **安全和隱私審查** - 評估數據處理和安全性

## 💬 Sample Trigger Prompts

### Scenario 1: Tool Evaluation
```
使用 ai-tool-review：
請評估 [工具名稱] 的適用性：
- 用途：[預期用途]
- 功能需求：[需要的功能]
- 性能要求：[性能指標]
- 預算限制：[預算]
- 使用場景：[應用場景]
```

### Scenario 2: Comparative Analysis
```
使用 ai-tool-review：
比較以下AI工具在 [任務] 上的性能：
- 工具列表：[工具1, 工具2, 工具3]
- 評估標準：[標準]
- 用例場景：[場景]
- 期望成本：[預算]
- 集成要求：[要求]
```

### Scenario 3: Security & Privacy Review
```
使用 ai-tool-review：
對 [工具名稱] 進行安全和隱私審查：
- 數據處理方式
- 合規性檢查（GDPR等）
- 加密和認證方式
- 隱私政策評估
- 風險識別
```

## 📋 Output Format

✅ 應該包含：
- 工具概述和功能清單
- 性能指標和基準測試結果
- 優缺點分析
- 成本分析
- 集成兼容性評估
- 安全性評分
- 推薦和結論
- 替代方案
