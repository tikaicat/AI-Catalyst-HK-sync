# Generic DOCX Processor

Generate, process, and convert Word documents (.docx format).

## When to Use
- Generating Word documents from data or templates
- Converting content into professional .docx format
- Batch editing existing Word documents programmatically
- Exporting reports or analysis as Word files

## Process

1. **Clarify content** - What goes in the document? Headers, body, tables, images?
2. **Define structure** - Outline sections and formatting requirements
3. **Choose method** - python-docx library (recommended) or pandoc
4. **Generate code** - Write the script to produce the .docx
5. **Validate output** - Confirm formatting matches requirements

## Output Should Include
- Python script using python-docx (or equivalent)
- Document structure with proper heading levels
- Table generation if needed
- Styling and formatting instructions
- Save path and filename convention
- Instructions to run the script
