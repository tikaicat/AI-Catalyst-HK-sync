# Generic PPTX Processor

Create, generate, and edit PowerPoint presentations (.pptx format).

## When to Use
- Auto-generating slides from an outline or dataset
- Creating reusable presentation templates
- Batch-updating content across multiple slides
- Exporting reports or summaries as PowerPoint

## Process

1. **Define content** - What is the outline? How many slides?
2. **Set design** - Theme, fonts, colours, layout preferences
3. **Use python-pptx** - Recommended library for .pptx generation
4. **Build slides** - Title, content, images, charts slide by slide
5. **Export file** - Save with consistent naming convention

## Output Should Include
- Python script using python-pptx
- Slide layout definitions (title slide, content slide, etc.)
- Text placeholder population
- Chart or image insertion if needed
- Theme/colour configuration
- Run instructions and dependency installation
